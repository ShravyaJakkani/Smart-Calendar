

# 📅 Smart Calendar

## 🚀 Project Overview

Smart Calendar is a simple and interactive web application designed to help users manage their daily tasks efficiently. It provides a clean calendar interface where users can select dates, add tasks, and organize their schedule in a user-friendly way.

---

## 💡 Idea Behind the Project

The main idea of this project is to create a **minimal and visually appealing productivity tool** that allows users to:

* Track their daily activities
* Stay organized
* Improve time management

Unlike complex tools, Smart Calendar focuses on **simplicity, clarity, and ease of use**.

---

## ✨ Features

* 📌 Select and highlight dates
* 📝 Add tasks for specific dates
* 📖 Notes/Task section for better planning
* 🖼️ Dynamic hero image (user upload / random image)
* ⚠️ Warning message for incomplete tasks at the end of the day
* 📤 Export tasks as a readable summary
* 📱 Responsive UI (works on mobile & desktop)

---

## 🛠️ Technologies Used

* React.js
* Tailwind CSS
* JavaScript

## 📌 How to Use

1. Select a date from the calendar
2. Add tasks for that day
3. View and manage your schedule
4. Track incomplete tasks with alerts
5. Export your schedule if needed
